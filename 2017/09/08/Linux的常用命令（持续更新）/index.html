Linux的常用命令行

基本入门操作

cd        --更换当前目录或文件夹
rm        --删除文件（删除目录加  -r 控制符）
mkdir        --创建文件夹
rmdir        --删除文件夹（只能是空的）
pwd        --显示当前目录
cp        --复制文件
ls        --显示当前目录下的文件
mvdir        --移动文件夹（可用作重命名操作）
